# DETRACTOR

Mini web app statica ispirata alle recensioni negative Apple Store. Non richiede build tools: basta aprire `index.html` oppure pubblicare i file su un hosting statico.

## File

- `index.html`: struttura della pagina
- `style.css`: tema dark, layout e animazioni
- `script.js`: logica slot, popup e recensioni casuali

## Come provarla in locale

1. Apri la cartella in un editor.
2. Fai doppio clic su `index.html` oppure trascinalo nel browser.
3. Premi `Tira la leva e tenta la fortuna`.

## Immagine iniziale

La home usa gia il file `cover.png`, cioe il poster che hai fornito. Se vorrai sostituirlo piu avanti:

1. rimpiazza `cover.png` con un altro file
2. oppure modifica il percorso immagine dentro `.home-image` in `style.css`

## Come condividerla

### Netlify Drop

1. Vai su [Netlify Drop](https://app.netlify.com/drop).
2. Trascina l'intera cartella.
3. Ottieni subito un link pubblico da mandare agli amici.

### GitHub Pages

1. Carica questi file in un repository GitHub.
2. Attiva GitHub Pages dalle impostazioni del repository.
3. Condividi l'URL generato.

### Vercel

1. Importa la cartella o il repository su Vercel.
2. Pubblica come sito statico.
3. Condividi il link.
